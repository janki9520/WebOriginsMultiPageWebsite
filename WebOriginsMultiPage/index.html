Origin of Internet

The Internet is the global system of interconnected computer networks that use TCP/IP protocol suite to link billions of devices worldwide.

It was conceived by the Advanced Research Projects Agency (ARPA) of the U.S. government in 1969 and was first known as the ARPANet.

Internet is used as a “BACKBONE” for other online applications, including Email, Instant Messaging, Skype, Streaming Video, and the World Wide Web.

Internet vs WWW

Web & Internet are two seperate but related things (Web!=Internet).

The Internet is a massive network of networks, a networking infrastructure. It connects millions of computers together globally, forming a network in which any computer can communicate with any other computer as long as they are both connected to the Internet.

Whereas, the World Wide Web, or simply Web, is a way of accessing information over the medium of the Internet. It is an information-sharing model that is built on top of the Internet. The Web uses the HTTP protocol, only one of the languages spoken over the Internet, to transmit data. Web services, which use HTTP to allow applications to communicate in order to exchange business logic, use the the Web to share information.

Origin of WWW

World Wide Web(www) is a system of interlinked hypertext documents connected by the internet.

Tim Berners-Lee, a British scientist at CERN, invented the World Wide Web (WWW) in 1989.

The web was originally conceived and developed to meet the demand for automatic information-sharing between scientists in universities and institutes around the world.

Tim suggested three main technologies that meant all computers could understand each other (HTML, URL and HTTP). All of these remain in use today.

Internet Technologies

Email

E-mail (electronic mail) is the exchange of computer-stored messages by telecommunication.

Examples of email includes; Gmail, Yahoo, Microsoft Outlook, Hotmail etc.

Telnet

Telnet is a user command and an underlying TCP/IP protocol for accessing remote computers.

Through Telnet, an administrator or another user can access someone else's computer remotely.

SSH

SSH, also known as Secure Socket Shell, is a network protocol that provides administrators with a secure way to access a remote computer.

FTP

File Transfer Protocol (FTP) is a standard Internet protocol for transmitting files between computers on the Internet over TCP/IP connections.

HTTP

HyperText Transfer Protocol(HTTP) is the underlying protocol used by the World Wide Web and this protocol defines how messages are formatted and transmitted, and what actions Web servers and browsers should take in response to various commands.

Gopher

From about 1992 through 1996, Gopher was an Internet application in which hierarchically-organized text files could be brought from servers all over the world to a viewer on your computer.

Browser

A browser is an application program that provides a way to look at and interact with all the information on the World Wide Web.

Examples of web browsers includes; Chrome, Mozilla, Mosaic, Safari etc.

Lynx

Lynx is a keyboard-oriented text-only Web browser that was developed at the University of Kansas primarily for students who used UNIX workstations.

Mosaic

Marc Andreessen and his team invented Mosaic (original NCSA page), the first popular Web browser, which greatly helped spread use and knowledge of the web across the world.

Hypertext

Hypertext is text that is not constrained to be linear but rather is text that contains links to other text

Hypermedia is a term for Hypertext not constrained to be text such as graphics, video, and sound
