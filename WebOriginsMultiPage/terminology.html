Definitions

  1. Internet: It is a computer network consisting of interconnected networks which use TCP/IP suite to provide a variety of information and communication facilities to the users.

  2. World Wide Web: It is a network of online content or a collection of internet resources(such as; telnet, FTP, Video files, Web pages etc.) that is formatted in HTML and accessed via HTTP over the internet.

  3. Hypertext: It is a text that is linked to a hyperlink which allows user to move to a different content either on the same page or on the different page.

  4. Multimedia: It is a form of presenting information using a combination of different content forms such as; Text, Audio, Images, Animations, Video & various other interactive contents.

  5. CLI: It refers to "Command Line Interface" to which the user responds to a visual prompt by typing in a command on a specified line, and receives a respond back from the system.

  6. GUI: It stands for "Graphical User Interface" which is a type of user interface that allow users to interact with the electronic devices through graphical icons and various visual indicators.

  7. HCI: It refers to "Human Computer Interacation" which is a study of how user interacts with computers and to what extent computers are capable enough for this interaction.

  8. FTP:FTP stands for "File Transfer Protocol" which allows user to transfer files between a client and a server on a computer network over TCP/IP connections.

  9. SSH: SSH(Secure Socket Shell/Secure Shell) is a UNIX based cryptographic network protocol for operating network services securely over an unsecured network.

  10. HTTP: "Hypertext Transfer Protocol(HTTP)" is an application protocol which provides a set of rules for transferring files(Text, Graphics, Images, Sound etc.) on the world wide web between a client and a server.

  11. Telnet: It is a user command and an underlying network protocol(TCP?IP) for accessing remote computers.

  12. HTML: It refers to "Hypertext Markup Language" which is a standardized system for marking symbols or codes in a file intended for display on the world wide web.

  13. CSS: Cascading Style Sheet is the language for describing the presentation of web pages, including colors, layout, fonts etc.
